# Laboratorio DMZ con Cisco Packet Tracer

Este repositorio contiene la implementación de una DMZ segura utilizando Cisco Packet Tracer.

## Contenido
- `Informe_DMZ_Laboratorio.md`: informe completo del laboratorio.
- `evidencias/`: capturas de pantalla de las pruebas realizadas.
- Archivo `.pkt`: topología final del laboratorio.

## Objetivo
Configurar una red segmentada con LAN, DMZ y red externa, aplicando NAT y ACLs para controlar el acceso a los servicios y proteger la red interna.
